# Архитектурные решения в программном обеспечении

### Лабораторная работа №1  
Node.js приложение для получения IP-адреса через внешние API (ip-api.com или jsonip.com).
Выбор API осуществляется через переменную окружения `TYPE`.

#### Запуск
`npm install`
`TYPE=ipapi node index.js`

### Лабораторная работа №2
Докеризация приложения из ЛР1 с использованием Docker и docker-compose.

#### Запуск
`docker-compose up --build`

### Лабораторная работа №3
Подготовлены Kubernetes-манифесты для развёртывания приложения: Deployment, Service, Ingress.
